# Locations

Locations support combat readability on one shared field with depth. Walkable space must communicate foreground/background separation clearly and leave enough horizontal room for melee, runs, and projectile play.

Factory location direction:
- Hard floor edges and readable depth bands.
- Industrial props that do not hide hitboxes.
- Hazards only when a task defines damage, timing, and spatial interaction.
- Background detail stays grounded and does not compete with combat silhouettes.
