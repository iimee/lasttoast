# Scenario

Папка собрана как русский пакет сценарных и уровневых материалов для быстрого доступа.

## Состав
- `LAST_TOAST_scenario_v2.md` — основной сценарный каркас игры, скопирован из корня проекта.
- `level1_factory_setpieces_ru.md` — русский документ по сетписам завода, скопирован из `docs/`.
- `level2_stadium_design_ru.md` — перевод стадиона.
- `level3_bridge_design_ru.md` — перевод моста.
- `level4_metro_design_ru.md` — перевод метро.
- `level5_forest_camp_design_ru.md` — перевод лесного лагеря.
